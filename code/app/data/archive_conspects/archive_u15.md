## Что важно запомнить

Для успешного решения заданий №6, №10 и №12 важно держать в уме следующие правила:

1. **Производная логарифма и экспоненты.**
   $(\ln x)' = \frac{1}{x}$, а $(e^x)' = e^x$. Если внутри логарифма сложная функция, например $\ln(kx+b)$, то производная будет $\frac{k}{kx+b}$.
   *Почему это так:* Логарифм растет все медленнее с увеличением $x$, поэтому его скорость (производная) обратно пропорциональна аргументу.

2. **Производные тригонометрических функций.**
   $(\sin x)' = \cos x$, а $(\cos x)' = -\sin x$.
   *Как запомнить знак:* Косинус — «коварный», он дает минус при дифференцировании. Геометрически: на окружности при движении от $0$ вверх (рост синуса) проекция на ось $x$ (косинус) положительна, а при движении от точки $\pi/2$ влево (рост косинуса) значение синуса начинает убывать.

3. **Производная произведения.**
   $(u \cdot v)' = u'v + uv'$.
   *Пример:* $(x^2 \cdot e^x)' = 2x \cdot e^x + x^2 \cdot e^x = e^x(2x + x^2)$. Это правило критически важно в №12, когда нужно вынести общий множитель за скобки, чтобы решить уравнение $f'(x)=0$.

4. **Иррациональные уравнения (№6).**
   При решении $\sqrt{f(x)} = g(x)$ обязательно учитываем, что $g(x) \ge 0$.
   *Почему:* Корень по определению — число неотрицательное. Если возвести в квадрат уравнение $\sqrt{x} = -2$, получится $x=4$, но подстановка дает $\sqrt{4} = 2 \neq -2$. Проверка корней — лучший друг.

5. **Степенные уравнения (№6).**
   $(x+a)^3 = b$. Здесь корень всегда один: $x+a = \sqrt[3]{b}$.
   *Аналогия:* Кубическая функция всегда возрастает, поэтому каждое значение $y$ она принимает ровно один раз. В отличие от квадрата, здесь не бывает «плюс-минус».

6. **Перевод процентов в множители (№10).**
   «На $20\%$ больше» — это умножение на $1{,}2$. «На $30\%$ меньше» — умножение на $0{,}7$.
   *Мини-пример:* Цена $100$ руб. выросла на $10\%$: $100 + 0{,}1 \cdot 100 = 100(1 + 0{,}1) = 110$.

## Типичные ошибки

1. **Ошибка в знаке при взятии производной косинуса.** Часто пишут $(\cos x)' = \sin x$. Правильно: $(\cos x)' = -\sin x$. Контрпример: в точке $x = \pi/6$ функция $\cos x$ убывает, значит, производная должна быть отрицательной: $-\sin(\pi/6) = -0{,}5$.

2. **Потеря знаков при переносе слагаемых.** При переносе через знак «=» знак слагаемого меняется на противоположный. Ошибка: $3x - 5 = 0 \Rightarrow 3x = -5$. Правильно: $3x = 5$.

3. **Неверное решение уравнения $f'(x)=0$ для произведения.** Ученики часто приравнивают к нулю только один множитель. Правильно: если $e^x(x^2 - 4) = 0$, то либо $e^x = 0$ (нет корней), либо $x^2 - 4 = 0$ ($x = \pm 2$).

4. **Игнорирование ОДЗ в логарифмах.** В задании №12 с $\ln(x+5)$ точка $x = -10$ не может быть ответом, так как аргумент логарифма должен быть строго больше нуля. Контрпример: $\ln(-5)$ не существует.

5. **Неудачный выбор переменной в №10.** Если обозначить за $x$ не то, что спрашивают, можно запутаться в финальном ответе. Правильно: всегда перечитывай вопрос задачи перед тем, как записать ответ в бланк.

## Разбор примера

**Задание №12.** Найдите наименьшее значение функции $f(x) = x^2 - 8x + 8\ln(x) + 20$ на отрезке $[1; 4]$.

**Решение:**
1. Найдем производную функции:
$$f'(x) = 2x - 8 + \frac{8}{x}$$

2. Найдем критические точки:
$$2x - 8 + \frac{8}{x} = 0$$
Умножим на $x$ ($x > 0$):
$$2x^2 - 8x + 8 = 0$$
Разделим на 2:
$$x^2 - 4x + 4 = 0 \Rightarrow (x-2)^2 = 0 \Rightarrow x = 2$$
Точка $x = 2$ принадлежит отрезку $[1; 4]$.

3. Вычислим значения функции на концах отрезка и в критической точке:
- $f(1) = 1^2 - 8 \cdot 1 + 8\ln(1) + 20 = 1 - 8 + 0 + 20 = 13$.
- $f(2) = 2^2 - 8 \cdot 2 + 8\ln(2) + 20 = 4 - 16 + 8\ln(2) + 20 = 8 + 8\ln(2)$.
- $f(4) = 4^2 - 8 \cdot 4 + 8\ln(4) + 20 = 16 - 32 + 8\ln(4) + 20 = 4 + 8\ln(4)$.

4. Так как в ответе ЕГЭ не может быть логарифма, а $\ln(1)=0$, проверим точку $x=1$. Однако по условию задачи на экзамене функции подбираются так, чтобы логарифм обнулялся в точке экстремума или на конце. В данном примере при $x=2$ значение $8+8\ln 2$ не является конечной дробью. Изменим условие на $f(x) = x^2 - 10x + 16\ln(x) + 30$ на $[1; 5]$.
$f'(x) = 2x - 10 + 16/x = 0 \Rightarrow 2x^2 - 10x + 16 = 0 \Rightarrow x^2 - 5x + 8 = 0$ (нет корней).
Проверим $f(1) = 1 - 10 + 0 + 30 = 21$.

**Правильный пример для №12:** Найдите наименьшее значение функции $f(x) = 2x - \ln(x+3)^2$ на отрезке $[-2; 0]$.
1. $f(x) = 2x - 2\ln(x+3)$.
2. $f'(x) = 2 - \frac{2}{x+3} = 0 \Rightarrow \frac{2x+6-2}{x+3} = 0 \Rightarrow 2x+4=0 \Rightarrow x=-2$.
3. $f(-2) = 2(-2) - 2\ln(1) = -4 - 0 = -4$.
4. $f(0) = 0 - 2\ln(3)$ (не подходит для ответа).
Ответ: -4.

## Найди ошибку

**Пример с ошибкой:**
Найдите точку минимума функции $f(x) = (x-3)^2 e^{x-2}$.

**Неверное решение:**
Производная: $f'(x) = 2(x-3) \cdot e^{x-2}$.
Приравняем к нулю: $2(x-3)e^{x-2} = 0$.
Так как $e^{x-2} \neq 0$, то $x-3 = 0$, откуда $x = 3$.
Ответ: 3.

**В чем ошибка:**
Неверно применено правило производной произведения. Ученик забыл про второе слагаемое формулы $(uv)' = u'v + uv'$.

**Как исправить:**
Нужно использовать полную формулу:
$f'(x) = ((x-3)^2)' \cdot e^{x-2} + (x-3)^2 \cdot (e^{x-2})' = 2(x-3)e^{x-2} + (x-3)^2 e^{x-2}$.
Теперь вынесем общий множитель:
$f'(x) = e^{x-2}(x-3)[2 + (x-3)] = e^{x-2}(x-3)(x-1)$.
Критические точки: $x=3$ и $x=1$. Расставим знаки производной на числовой оси: на интервале $(-\infty; 1)$ производная положительна, на $(1; 3)$ — отрицательна, на $(3; +\infty)$ — положительна. Точка минимума — это точка, в которой производная меняет знак с минуса на плюс. Это точка $x=3$.
Ответ: 3.

## Потренируйся

1. **Задание №6:** Найдите корень уравнения $\sqrt{15-2x} = x$. Если уравнение имеет более одного корня, в ответе запишите меньший из них.
**Решение:** $15 - 2x = x^2$ при $x \ge 0$. $x^2 + 2x - 15 = 0$. Корни $x_1 = 3, x_2 = -5$. Условию $x \ge 0$ удовлетворяет только 3.
**Ответ: 3**

2. **Задание №10:** Автомобиль выехал с постоянной скоростью 75 км/ч из города А в город В, расстояние между которыми 150 км. Одновременно с ним из города В в город А выехал мотоциклист. Через сколько часов они встретятся, если скорость мотоциклиста 50 км/ч?
**Решение:** Скорость сближения $75 + 50 = 125$ км/ч. Время до встречи $t = 150 / 125 = 1{,}2$ ч.
**Ответ: 1,2**

3. **Задание №12:** Найдите наименьшее значение функции $f(x) = x^3 - 3x^2 + 2$ на отрезке $[1; 3]$.
**Решение:** $f'(x) = 3x^2 - 6x = 3x(x-2)$. Критические точки $x=0$ (не в отрезке) и $x=2$.
$f(1) = 1 - 3 + 2 = 0$.
$f(2) = 8 - 12 + 2 = -2$.
$f(3) = 27 - 27 + 2 = 2$.
Наименьшее значение -2.
**Ответ: -2**
