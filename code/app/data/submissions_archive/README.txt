ДЕПЕРСОНАЛИЗАЦИЯ (152-ФЗ)
-------------------------
Все персональные данные удалены из экспорта.
Вместо ФИО, email и прочих идентификаторов используется
анонимный порядковый номер ученика (user_number).
Из профиля ученика сохранён только класс (grade).

ФАЙЛЫ
-----
1. micromodules_competencies.csv
   Справочник: микромодули (ноды) и связанные компетенции.
   Не содержит данных учеников.

2. user_practice_submissions.csv
   Практики (КИМы и блоки), решённые учениками.
   user_number — анонимный номер ученика, grade — класс.

3. user_task_submissions.csv
   Отдельные задания внутри практик.
   user_number — анонимный номер ученика.
   is_correct = True, если набранный балл >= максимального.

4. task_competence_mapping.csv
   Справочник: задания и связанные компетенции.
   Не содержит данных учеников.

5. user_competences.csv
   Оценки компетенций учеников (вероятность владения).
   user_number — анонимный номер ученика.
   probability — вероятность владения компетенцией (0..1).
