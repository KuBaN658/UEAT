# UEAT

Расшифровывается как USE Exam AI Tutor (подготовка к ЕГЭ)

Monorepo для проекта генерации персонализированных учебных конспектов.

## О проекте

**Название (RU):** Генерация персонализированных учебных конспектов по предметам ЕГЭ на основе языковых моделей и пользовательской аналитики

**Название (EN):** Generation of Personalized Educational Notes for Unified State Exam Subjects Based on Language Models and User Analytics

**Руководитель:** Ахтямов Павел Ибрагимович

**Студенты:**
- Ахундов Алексей Назимович
- Жарковский Дмитрий Андреевич

**Тип:** Программный проект (3-4 человека)

### Цель

Разработать систему генерации учебных конспектов с использованием больших языковых моделей (LLM), обеспечивающую персонализированное представление теоретического материала на основе анализа данных об успеваемости и типичных ошибках пользователей.

### Задачи

- Написание контекстного сервера (MCP) для модели компетенций
- Обработка обратной связи слушателей и аналитики больших данных
- Написание обертки в виде веб-сервиса
- Проверка гипотез: ручная генерация конспектов против генерации при помощи ИИ
