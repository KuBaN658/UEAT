# UEAT — USE Exam AI Tutor

Репозиторий выпускной квалификационной работы по направлению «Программный проект».

**Название:** Генерация персонализированных учебных конспектов по предметам ЕГЭ на основе языковых моделей и пользовательской аналитики

**Руководитель:** Ахтямов Павел Ибрагимович

**Авторы:** Ахундов Алексей Назимович, Жарковский Дмитрий Андреевич

## Структура репозитория

```
├── doc/          — LaTeX-исходники ВКР и библиография
│   ├── src/vkr/  — основной текст (diplomawork.tex)
│   ├── build/    — собранные PDF (генерируются CI)
│   └── pdfs/     — публичные PDF-артефакты
└── code/         — исходный код сервиса генерации конспектов
```

## О сервисе (`code/`)

RAG-сервис, интегрированный с платформой «ЕГЭ Прорыв». Принимает историю ответов ученика, извлекает релевантные учебные материалы из векторной базы знаний и генерирует персонализированный конспект с помощью LLM.

Основные компоненты:

| Компонент | Назначение |
|---|---|
| `app/` | FastAPI-приложение: API, воркеры, RAG-пайплайн |
| `admin/` | Симулятор платформы и панель наблюдаемости |
| `k8s/` | Kubernetes-манифесты (namespace `ege`) |
| `app/spec/` | OpenAPI и JSON Schema контракты интеграции |
| `app/rag/eval/` | Оценочный контур (DeepEval, бенчмарки retrieval) |

Подробности — в [`code/README.md`](code/README.md).
